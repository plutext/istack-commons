Welcome to the iStack Commons project. JAXP, JAXB, SAAJ, and JAX-WS projects are often used together by other projects (such as JDK, Glassfish, ...).

But at the same time, they are released independently, and those technologies don't necessarily have dependencies between them.

To promote code reuse between these projects, we host the common part in a separate project, so that when we ship individually, each technology can carry istack-commons-*.jar, and when the technologies are delivered together into e.g. JDK, the footprint can be reduced.
