Project Description

JAXP, JAXB, SAAJ, and JAX-WS projects are often used together by other projects (such as Mustang, Glassfish. But at the same time, they are released independently, and those technolgoies don't necessarily have dependencies between them.

To promote code reuse between these projects, we host the common part in a separate project, so that when we ship individually, we can each carry istack-commons.jar, and when e deliver together into another technology, we can reduce the footprint.
